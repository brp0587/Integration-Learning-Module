            <h2 class="section-title">
                3.1 & 3.2 Concept & Basic Integrals 
                <span class="gu">аӘёаӘӮаӘ•аӘІаӘЁаӘЁа«Ӣ аӘ–а«ҚаӘҜаӘҫаӘІ аӘ…аӘЁа«Ү аӘ®а«ӮаӘіаӘӯа«ӮаӘӨ аӘЁаӘҝаӘҜаӘ®а«Ӣ</span>
            </h2>
            <div class="bilingual-grid">
                <div class="en-text">
                    Integration is the reverse of differentiation. The standard working rule is Linearity: 
                    <br>\( \int [a f(x) + b g(x)] dx = a \int f(x) dx + b \int g(x) dx \)
                </div>
                <div class="gu-text">
                    аӘёаӘӮаӘ•аӘІаӘЁ аӘҸ аӘөаӘҝаӘ•аӘІаӘЁаӘЁа«Җ аӘүаӘІаӘҹа«Җ аӘӘа«ҚаӘ°аӘ•а«ҚаӘ°аӘҝаӘҜаӘҫ аӘӣа«Ү. аӘӨа«ҮаӘЁа«Ӣ аӘ®а«ҒаӘ–а«ҚаӘҜ аӘ•аӘҫаӘ°а«ҚаӘҜаӘЁаӘҝаӘҜаӘ® аӘёа«ҒаӘ°а«ҮаӘ–аӘӨаӘҫ аӘӣа«Ү: 
                    <br>\( \int [a f(x) + b g(x)] dx = a \int f(x) dx + b \int g(x) dx \)
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 1: Evaluate \(\int x^5 dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Use the power rule \(\int x^n dx = \frac{x^{n+1}}{n+1} + c\).
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Substitute \(n=5\).<br>
                    $$ I = \frac{x^{5+1}}{5+1} + c = \frac{x^6}{6} + c $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 2: Evaluate \(\int \frac{1}{x^2} dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Express as a negative power.<br>
                    $$ I = \int x^{-2} dx $$
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Apply power rule with \(n=-2\).<br>
                    $$ I = \frac{x^{-2+1}}{-2+1} + c = -\frac{1}{x} + c $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 3: Evaluate \(\int (x^3 + 5x^2 - 4) dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Apply Linearity / аӘёаӘ°аӘөаӘҫаӘіаӘҫаӘЁа«Ӣ аӘЁаӘҝаӘҜаӘ® аӘІаӘҫаӘ—а«Ғ аӘ•аӘ°а«Ӣ.<br>
                    $$ I = \int x^3 dx + 5\int x^2 dx - 4\int 1 dx $$
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Apply power rule.<br>
                    $$ I = \frac{x^4}{4} + 5\left(\frac{x^3}{3}\right) - 4x + c $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 4: Evaluate \(\int (e^x + \sin x + \sec^2 x) dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Separate the integrals using linearity.<br>
                    $$ I = \int e^x dx + \int \sin x dx + \int \sec^2 x dx $$
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Apply standard formulas directly from the table.<br>
                    $$ I = e^x + (-\cos x) + \tan x + c = e^x - \cos x + \tan x + c $$
                </div>
            </div>

        <div class="section">
            <h2 class="section-title">
                3.3 Method of Substitution 
                <span class="gu">аӘҶаӘҰа«ҮаӘ¶аӘЁа«Җ аӘ°а«ҖаӘӨ</span>
            </h2>

            <div class="example-box">
                <div class="example-title">Example 1: Evaluate \(\int \frac{\cos(\log x)}{x} dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Let \( \log x = t \). Differentiating gives \( \frac{1}{x} dx = dt \).
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Substitute into the integral.<br>
                    $$ I = \int \cos(t) dt $$
                </div>
                <div class="step">
                    <strong>Step 3:</strong> Integrate and replace \(t\).<br>
                    $$ I = \sin(t) + c = \sin(\log x) + c $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 2: Evaluate \(\int \frac{\cos\sqrt{x}}{2\sqrt{x}} dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Let \( \sqrt{x} = t \). Then \( \frac{1}{2\sqrt{x}} dx = dt \).
                </div>
                <div class="step">
                    <strong>Step 2:</strong> The integral becomes:<br>
                    $$ I = \int \cos(t) dt $$
                </div>
                <div class="step">
                    <strong>Step 3:</strong> $$ I = \sin(t) + c = \sin(\sqrt{x}) + c $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 3: Evaluate \(\int x \sin(x^2) dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Let \( x^2 = t \). Differentiating gives \( 2x dx = dt \implies x dx = \frac{dt}{2} \).
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Substitute into the integral.<br>
                    $$ I = \int \sin(t) \frac{dt}{2} = \frac{1}{2} \int \sin(t) dt $$
                </div>
                <div class="step">
                    <strong>Step 3:</strong> Integrate and replace \(t\).<br>
                    $$ I = \frac{1}{2} (-\cos t) + c = -\frac{1}{2} \cos(x^2) + c $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 4: Evaluate \(\int \frac{e^{\tan^{-1} x}}{1+x^2} dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Let \( \tan^{-1} x = t \). Then \( \frac{1}{1+x^2} dx = dt \).
                </div>
                <div class="step">
                    <strong>Step 2:</strong> The integral simplifies to:<br>
                    $$ I = \int e^t dt $$
                </div>
                <div class="step">
                    <strong>Step 3:</strong> Integrate and replace \(t\).<br>
                    $$ I = e^t + c = e^{\tan^{-1} x} + c $$
                </div>
            </div>
        </div>

        <div class="section">
            <h2 class="section-title">
                3.4 Integration By Parts 
                <span class="gu">аӘ–аӘӮаӘЎаӘ¶аӘғ аӘёаӘӮаӘ•аӘІаӘЁ</span>
            </h2>

            <div class="example-box">
                <div class="example-title">Example 1: Evaluate \(\int x \log x dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> By LIATE, \(u = \log x\) and \(v = x\).
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Apply the formula \( \int uv = u\int v - \int(u'\int v) \).<br>
                    $$ I = (\log x) \int x dx - \int \left( \frac{d}{dx}(\log x) \int x dx \right) dx $$
                </div>
                <div class="step">
                    <strong>Step 3:</strong> Execute integrals.<br>
                    $$ I = (\log x)\left(\frac{x^2}{2}\right) - \int \frac{1}{x} \left(\frac{x^2}{2}\right) dx $$
                    $$ I = \frac{x^2}{2} \log x - \frac{1}{2} \int x dx = \frac{x^2}{2} \log x - \frac{x^2}{4} + c $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 2: Evaluate \(\int x e^x dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> By LIATE, \(u = x\) (Algebraic) and \(v = e^x\) (Exponential).
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Apply formula.<br>
                    $$ I = x \int e^x dx - \int \left( \frac{d}{dx}(x) \int e^x dx \right) dx $$
                </div>
                <div class="step">
                    <strong>Step 3:</strong> Evaluate.<br>
                    $$ I = x e^x - \int (1 \cdot e^x) dx = x e^x - e^x + c = e^x(x - 1) + c $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 3: Evaluate \(\int x \sin x dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> By LIATE, \(u = x\) and \(v = \sin x\).
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Apply formula.<br>
                    $$ I = x \int \sin x dx - \int \left( 1 \cdot \int \sin x dx \right) dx $$
                </div>
                <div class="step">
                    <strong>Step 3:</strong> Evaluate.<br>
                    $$ I = x (-\cos x) - \int 1(-\cos x) dx = -x\cos x + \int \cos x dx $$
                    $$ I = -x\cos x + \sin x + c $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 4: Evaluate \(\int x^2 e^x dx\) (Requires two applications)</div>
                <div class="step">
                    <strong>Step 1:</strong> By LIATE, \(u = x^2\) and \(v = e^x\).<br>
                    $$ I = x^2 e^x - \int (2x \cdot e^x) dx $$
                </div>
                <div class="step">
                    <strong>Step 2:</strong> We must use By Parts again on \( \int 2x e^x dx \). Let \(u=2x\) and \(v=e^x\).<br>
                    $$ \int 2x e^x dx = 2x e^x - \int (2 \cdot e^x) dx = 2x e^x - 2e^x $$
                </div>
                <div class="step">
                    <strong>Step 3:</strong> Substitute back into Step 1.<br>
                    $$ I = x^2 e^x - (2x e^x - 2e^x) + c = e^x(x^2 - 2x + 2) + c $$
                </div>
            </div>
        </div>

        <div class="section">
            <h2 class="section-title">
                3.5 Definite Integral & Properties
                <span class="gu">аӘЁаӘҝаӘҜаӘӨ аӘёаӘӮаӘ•аӘІаӘҝаӘӨ аӘ…аӘЁа«Ү аӘ—а«ҒаӘЈаӘ§аӘ°а«ҚаӘ®а«Ӣ</span>
            </h2>

            <div class="example-box">
                <div class="example-title">Example 1: Evaluate \(\int_{0}^{1} e^x dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Integrate without limits first.<br>
                    $$ \int e^x dx = e^x $$
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Apply upper limit - lower limit.<br>
                    $$ I = [e^x]_0^1 = e^1 - e^0 = e - 1 $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 2: Evaluate \(\int_{-1}^{1} (x^2 + 1) dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Integrate term by term.<br>
                    $$ I = \left[ \frac{x^3}{3} + x \right]_{-1}^{1} $$
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Put limits: \( (\text{Upper}) - (\text{Lower}) \).<br>
                    $$ I = \left( \frac{1^3}{3} + 1 \right) - \left( \frac{(-1)^3}{3} + (-1) \right) $$
                    $$ I = \frac{4}{3} - \left(-\frac{4}{3}\right) = \frac{8}{3} $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 3: Evaluate \(\int_{0}^{\pi/2} \frac{\tan x}{\tan x + \cot x} dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Let this be Equation 1. Use Property \(\int_0^a f(x) dx = \int_0^a f(a-x) dx\).<br>
                    $$ I = \int_{0}^{\pi/2} \frac{\tan(\frac{\pi}{2}-x)}{\tan(\frac{\pi}{2}-x) + \cot(\frac{\pi}{2}-x)} dx $$
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Since \(\tan(\pi/2 - x) = \cot x\) and \(\cot(\pi/2 - x) = \tan x\):<br>
                    $$ I = \int_{0}^{\pi/2} \frac{\cot x}{\cot x + \tan x} dx \quad \dots (2) $$
                </div>
                <div class="step">
                    <strong>Step 3:</strong> Add Eq (1) and Eq (2).<br>
                    $$ 2I = \int_{0}^{\pi/2} \frac{\tan x + \cot x}{\tan x + \cot x} dx = \int_0^{\pi/2} 1 dx = [x]_0^{\pi/2} = \frac{\pi}{2} \implies I = \frac{\pi}{4} $$
                </div>
            </div>

            <div class="example-box">
                <div class="example-title">Example 4: Evaluate \(\int_{1}^{e} \frac{(\log x)^2}{x} dx\)</div>
                <div class="step">
                    <strong>Step 1:</strong> Substitution. Let \(\log x = t \implies \frac{1}{x} dx = dt\).
                </div>
                <div class="step">
                    <strong>Step 2:</strong> Change Limits: When \(x=1, t=\log(1)=0\). When \(x=e, t=\log(e)=1\).
                </div>
                <div class="step">
                    <strong>Step 3:</strong> Evaluate new integral.<br>
                    $$ I = \int_{0}^{1} t^2 dt = \left[ \frac{t^3}{3} \right]_0^1 = \frac{1^3}{3} - 0 = \frac{1}{3} $$
                </div>
            </div>
        </div>

        <div class="section">

        // Smart Input Pre-Processor for Student-friendly typing
        function formatStudentInput(str) {
            let s = str.toLowerCase();
            
            // 1. Inverse Trig Functions (sin^-1 x -> asin(x))
            s = s.replace(/sin\^-1\s*\(?/g, 'asin(');
            s = s.replace(/cos\^-1\s*\(?/g, 'acos(');
            s = s.replace(/tan\^-1\s*\(?/g, 'atan(');
            
            // Note: If they type sin^-1 x without closing parenthesis, 
            // nerdamer might still parse 'asin(x)' if we leave it as 'asin( x'. 
            // We rely on Nerdamer's flexibility here.
            
            // 2. Implicit multiplication (e.g. "x sin(x)" -> "x*sin(x)")
            // Match variable 'x' or number followed by space and a function name
            s = s.replace(/([x0-9])\s+(sin|cos|tan|log|ln|exp|asin|acos|atan)/g, '$1*$2');
            
            return s;
        }

        // Method Analyzer heuristics based on string matching
        function analyzeFunction(funcStr) {
            let methods = [];
            if(funcStr.includes('*') && (funcStr.includes('sin') || funcStr.includes('cos') || funcStr.includes('exp') || funcStr.includes('log') || funcStr.includes('e^'))) {
                methods.push("Product Rule / By Parts (LIATE) - аӘ–аӘӮаӘЎаӘ¶аӘғ аӘёаӘӮаӘ•аӘІаӘЁ");
            }
            if(funcStr.includes('/') && funcStr.includes('(') && funcStr.includes(')')) {
                methods.push("Rational / Partial Fractions (аӘҶаӘӮаӘ¶аӘҝаӘ• аӘ…аӘӘа«ӮаӘ°а«ҚаӘЈаӘҫаӘӮаӘ•) or Substitution");
            }
            if((funcStr.includes('sin') || funcStr.includes('cos') || funcStr.includes('tan')) && funcStr.includes('^')) {
                methods.push("Trigonometric Identities (аӘӨа«ҚаӘ°аӘҝаӘ•а«ӢаӘЈаӘ®аӘҝаӘӨа«ҖаӘҜ аӘЁаӘҝаӘӨа«ҚаӘҜаӘёаӘ®)");
            }
            if(funcStr.includes('log') || funcStr.includes('exp') || funcStr.includes('e^')) {
                methods.push("Substitution Method (аӘҶаӘҰа«ҮаӘ¶аӘЁа«Җ аӘ°а«ҖаӘӨ)");
            }
            if(methods.length === 0) {
                methods.push("Standard Polynomial Power Rules (аӘ®а«ӮаӘіаӘӯа«ӮаӘӨ аӘЁаӘҝаӘҜаӘ®а«Ӣ)");
            }
            return methods;
        }

        function calculateAdvancedIntegration() {
            const rawInput = document.getElementById('advanced-function').value;
            const type = document.getElementById('adv-calc-type').value;
            const lowerStr = document.getElementById('adv-lab-lower').value;
            const upperStr = document.getElementById('adv-lab-upper').value;
            const output = document.getElementById('solution-output');

            if (!rawInput.trim()) {
                output.innerHTML = "<p style='color:red;'>Please enter a function.</p>";
                output.style.display = "block";
                return;
            }

            try {
                // Pre-process user typing (e.g. sin^-1, x sinx)
                let mathFunc = formatStudentInput(rawInput);
                
                // 1. Clean formatting for LaTeX display
                let inputTex = nerdamer(mathFunc).toTeX();
                let limitsTex = type === 'definite' ? `_{${nerdamer(lowerStr).toTeX()}}^{${nerdamer(upperStr).toTeX()}}` : '';
                
                // 2. Analyze Method (using original input to catch e^x and spaces)
                let recommendedMethods = analyzeFunction(rawInput.toLowerCase());
                let badges = recommendedMethods.map(m => `<span class="analysis-badge">Strategy: ${m}</span>`).join('');

                // 3. Symbolic Integration via Nerdamer
                let integralResult = nerdamer(`integrate(${mathFunc}, x)`);
                let integralTex = integralResult.toTeX();

                let stepsHTML = `
                    <h3 style="color:var(--primary); margin-top:0;">Smart Symbolic Analysis:</h3>
                    <div style="margin-bottom:1.5rem;">${badges}</div>
                    
                    <div class="step">
                        <strong>Step 1: Setup Integral Form</strong><br>
                        $$ I = \\int${limitsTex} \\left( ${inputTex} \\right) dx $$
                    </div>

                    <div class="step">
                        <strong>Step 2: Apply Integration Rules & Execute</strong><br>
                        Applying algebraic evaluation to find the antiderivative:
                        $$ \\int \\left( ${inputTex} \\right) dx = ${integralTex} $$
                    </div>
                `;

                // 4. Definite limit evaluation if requested
                if (type === 'definite') {
                    // Evaluate at upper
                    let valUpper = nerdamer(integralResult.text(), {x: upperStr});
                    // Evaluate at lower
                    let valLower = nerdamer(integralResult.text(), {x: lowerStr});
                    
                    let finalAns = nerdamer(`${valUpper.text()} - (${valLower.text()})`).evaluate();
                    let finalTex = finalAns.toTeX();
                    let decVal = Number(finalAns.text()).toFixed(4);

                    stepsHTML += `
                    <div class="step">
                        <strong>Step 3: Apply Limits (Fundamental Theorem)</strong><br>
                        Substitute \(x = ${upperStr}\) and \(x = ${lowerStr}\).
                        $$ I = \\left[ ${integralTex} \\right]_{${lowerStr}}^{${upperStr}} $$
                        $$ I = (${valUpper.toTeX()}) - (${valLower.toTeX()}) $$
                    </div>
                    <div class="final-answer-box">
                        Final Exact Answer: $$ I = ${finalTex} $$
                        <span style="font-size:1rem; font-weight:normal; display:block; margin-top:5px;">Approx: ${decVal !== 'NaN' ? decVal : 'Symbolic Expression'}</span>
                    </div>
                    `;
                } else {
                    stepsHTML += `
                    <div class="final-answer-box">
                        Final Answer: $$ I = ${integralTex} + C $$
                    </div>
                    `;
                }

                output.innerHTML = stepsHTML;
                output.style.display = "block";

                if (window.MathJax) {
                    MathJax.typesetPromise([output]);
                }

            } catch (err) {
                console.error(err);
                output.innerHTML = `
                    <h3 style="color:red;">Calculation Error</h3>
                    <p>Could not parse or integrate the function. Please ensure correct mathematical syntax.</p>
                    <p><em>Tips: Ensure all brackets are closed. Inverse trig uses <code>sin^-1(x)</code> or <code>asin(x)</code>.</em></p>
                `;
                output.style.display = "block";
            }
        }
